# 人人影视客户端 for Flutter

### 功能一览

- 安卓端边下边播
- 自动适应DarkMode
- 本地播放
- 我的收藏
- 影视搜索
- 用户登录

- 支持全平台：Android, IOS, Windows, MAC OS, Web Server.
- 构建请参考Flutter官方文档。
- 目前仅支持安卓端边下边播，其他端仅支持其他方式下载。
